A desktop RPG-style character inventory management system built using C#, WPF and .NET.

The application simulates a character inventory with item management, equipment slots and dynamic stat updates.

Features:
-Equip and unequip items
-Character stat updates based on equipped items
- Real-time UI updates using data binding
